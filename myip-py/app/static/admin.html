自定义 Provider

按步骤填写：先保存元数据，再测试字段提取预览。启用并保存后可参与后台 /api/admin/lookup；进入公开 /api/ip 还需要显式允许。

步骤 1：基本信息

Provider id 用小写字母、数字和短横线；示例：example-risk-api。

步骤 2：Endpoint

Endpoint 必须是 HTTPS，并用 {ip} 表示查询 IP；示例：https://api.example.com/{ip}。

步骤 2.5：认证方式

有些 Provider 不需要认证，有些需要 API Key 或 Bearer Token。Key / Token 只保存在后台配置；预览和公开调用是否真正使用它，后续再接入请求头/参数。

步骤 3：字段映射
