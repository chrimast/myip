自定义 Provider

按步骤填写：先保存元数据，再测试字段提取预览。启用并保存后可参与后台 /api/admin/lookup；进入公开 /api/ip 还需要显式允许。

步骤 1：基本信息

Provider id 用小写字母、数字和短横线；示例：example-risk-api。

步骤 2：Endpoint

Endpoint 必须是 HTTPS，并用 {ip} 表示查询 IP；示例：https://api.example.com/{ip}。

步骤 2.5：认证方式

有些 Provider 不需要认证，有些需要 API Key 或 Bearer Token。Key / Token 只保存在后台配置；测试与公开调用会按认证方式携带请求头，导出和页面 JSON 会脱敏。

步骤 3：字段映射

JSON 校验会在提交前执行；示例：{"country":["location.country"]}。

步骤 4：测试验证

JSON 字段选择器

测试成功后会列出返回 JSON 的叶子路径；点击路径可复制到字段映射输入框。
